# 🏫 Class Charts Timetable

A custom integration for Home Assistant that tracks your school lessons, teachers, and classrooms in real-time.

---

### ✨ Key Features
* 📅 **Live Calendar:** View your full school week directly in the Home Assistant Calendar.
* 🔔 **Lesson Sensors:** Entities for `Current Lesson` and `Next Lesson`.
* 📍 **Room Tracking:** See exactly which classroom you need to be in.
* 👔 **Teacher Info:** Know who is teaching each session at a glance.

---

### 🚀 Getting Started
1. Install this integration via HACS.
2. Restart Home Assistant.
3. Go to **Settings > Devices & Services > Add Integration**.
4. Search for **Class Charts Timetable** and enter your login details.

---

### 📊 Dashboard Tips
Use a **Tile Card** or **Mushroom Card** to display your current lesson. You can set the icon to `mdi:school` and pick a blue color to match the Class Charts branding!
